DIRECTIONS

1. Open Spatial Analysis Starter.twbx.

2. From the Toronto Votes worksheet, create a map by dragging Geometry to Detail. Notice that Tableau uses COLLECT to aggregate the marks.

3. Add both Ward and Area Name to Detail.

4. Color the shapes by % Voted.

5. Add Number Voted to the Tooltip.

6. Modify the Tooltip to the following:


<Area Name>


Ward: <Ward>

 

<SUM(% Voted)> Voted
