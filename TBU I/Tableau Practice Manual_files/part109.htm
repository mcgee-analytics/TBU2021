DIRECTIONS

1. Open Background Images Starter.twbx.

2. From the Map menu, select Background Images, and then click NBA Savant – Steph Curry Shots. This will relate the image to the correct data set.

3. From the Background Images dialog box, click Add Image to add the background image.

4. Use the following settings to define the background image and the scale to be used when placing the marks on the image. The scale was defined by the user to match the data.
