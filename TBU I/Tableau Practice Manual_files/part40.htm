DIRECTIONS

1. Using the Date dimension, create a Custom date with a value of years, change to Date Part, and name it “Date (Discrete Years)”.

2. Create a hierarchy using Date (Discrete Quarters) and Date (Discrete Years) and name the new hierarchy “Date (Discrete Years to Quarters)”.

3. On the sheet named Year to Quarter Custom Date Hierarchy, drag the new hierarchy to the view on top of the current Month(Date) field in Columns.

4. Change the mark type to Bar.

Notice that you can now drill up and down between Year and Quarter. Using this type of hierarchies’ limits what you allow the user to view
