Customize Your Story Points


1. Make sure each story point is set to the state that you want by checking the following:

 Filter and parameter values

 Selections

 Sorts

 Pan and zoom (for maps)

In the example, these story points are customized as follows:


 Map Dashboard (1st instance) – filtered to magnitude 7.0 and above

 Map Dashboard (2nd instance) – filtered to magnitude 8.0 and above

 Timeline by region dashboard (2nd instance) – filtered to magnitude 5.0 and above.
