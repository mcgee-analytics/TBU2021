View One: Continuous Custom Dates


Create a line chart that shows profit by date using a custom continuous date in quarters.


DIRECTIONS

1. Open Custom Dates Starter.twbx.

2. Using the Date dimension, create a Custom date with a Date Value of quarters and name it “Date (Continuous Quarters)”.

3. On the sheet named Continuous Quarters, drag the newly created Date (Continuous Quarters) to

Columns on top of MONTH(Date) field to replace it entirely.
