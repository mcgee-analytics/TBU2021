Use a Parameter to Create a Dynamic Dimension


You would like to create a visualization that shows the data broken out by the selected date dimension. Create a parameter to allow the users to select the date dimension to use to display the data.



DIRECTIONS

1. On the Using Parameter to Select Date Dimension worksheet, create a line graph showing SUM(Sales)

by Order Date.

2. Create a new parameter with the following settings:

 Name = Set the Date Interval

 Data Type = Integer

 Display Format= Automatic

 Allowable Values = List

 List of Values:

 

Value Display As
