DIRECTIONS

1. Open Moving Average Starter.twbx.

2. Create the initial view, a line chart of Profit by Order Date.

3. Change the Order Date on the Columns shelf to a continuous Month.

4. Right-click on the Profit field and select Add Table Calculation to show Profit as a Moving Average

using the following options:

 

Calculation Type: Moving Calculation Summarize values using: Average Previous values: 4Compute Using: Table (across)
