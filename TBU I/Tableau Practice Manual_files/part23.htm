View One


The data source contains historical data for the players on the rosters for all teams in the 2018 World Cup. Create a view to show the total goals by player. Filter the view to show only those players that are forwards, with a type of Cap, who scored at least 30 goals.


DIRECTIONS

1. Open Using Filters Starter.twbx.

2. On the Goals by Player worksheet, build a view to compare the Goals for each Player. Drag Goals to

Columns and Name to Rows.

3. Sort the Name by Goals in descending order, using the sort icon on the menu bar.

4. Add a Dimension Filter to Include only forwards (Use Position - FW).

5. Add a Measure Filter to show only those players with at least 30 goals (Use SUM(Goals)).

6. Show the filter and Edit the Title to “Goals”.
