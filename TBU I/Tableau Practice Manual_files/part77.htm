Practice: Using Join Calculations

You have a workbook that is connected to a data source with information about when members have joined a club. Set up a cross database join, using a join calculation, to a second data source that has sales information on each member of the club.


Then, create a bar chart so you can analyze how long the members with the greatest sales has been with the club.
