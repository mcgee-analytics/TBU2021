DIRECTIONS

1. Open Dual Axis Chart Starter.twbx.

2. Create an initial view showing Sales by Date.

3. Change the Year(Date) on Columns to continuous Month (for example May 2015)

4. Change the mark type to Bar.

5. Create a dual axis chart using Budget Sales. Hint: Use the opposite axis.

6. Change the mark type to Line for the newly created Budget Sales axis.

7. Synchronize the Budget Sales axis to the Sales axis.

8. Edit the colors so Sales is shown as light gray bars. Hint: use the Tableau Classic 20 color palette.
